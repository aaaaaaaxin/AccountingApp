# 家庭记账应用需求文档 v1.2

## 1. 需求概述
- 一句话描述：对家庭记账应用进行UI重构，采用现代化的底部导航栏和卡片式布局，优化用户体验
- 目标用户：家庭用户
- 使用场景：日常记账、查看统计、管理账户

## 2. 功能需求
### 2.1 核心功能
- `[ ]` 底部导航栏：首页、报表、统计、账户四个标签页（记账为二级页面入口）
- `[ ]` 首页重构：展示本月收支统计概览、快速入口
  - 顶部栏固定在顶部（信息架构一致性）：所有底部导航主页面（首页/记账/统计/账户/报表）均保持顶部栏固定
  - 顶部栏显示当前账本与当前页面标题
  - 首页顶部栏不提供“创建账本”入口（创建账本入口统一收敛到“账本选择/管理”弹层或账户页）
  - 记录列表按日期分组展示，并展示当日收入/支出与当日结余小计
  - 列表项信息展示规则：备注最多1行省略；标签最多展示3个（超出省略）
- `[ ]` 记账页面（v1.2默认形态）：作为二级页面打开与关闭，不占用底部导航tab
  - 入口：底部导航中间主按钮“记账”打开记账页
  - 关闭：点击右上角关闭按钮（或返回）关闭记账页并返回打开前页签
  - 场景：新增记账与编辑记账均复用该二级页面
  - 能力：支出/收入切换、分类网格选择（支持子分类）、支付方式、日期快捷项、模版、标签、数字键盘
  - 布局（截图版拆解）：
    - 顶部栏（固定）：
      - 左侧：当前账本入口（下拉/弹层），用于切换账本
      - 左侧区域：支出/收入分段切换（Segmented），位置随账本入口联动
      - 右侧：关闭按钮（X），关闭后返回打开前页签
    - 分类区（可滚动区域的上半部分）：
      - 标题行：左侧“分类”，右侧展示当前选择状态（未选择/已选择）
      - 分类管理：分类区提供“分类管理”入口，进入分类明细页
      - 主分类网格：图标+名称的网格按钮；点击选中主分类
      - 选中态：选中主分类高亮；右侧状态显示随选中变化
    - 子分类区（主分类下）：
      - 展示当前主分类的子分类入口与列表（若有）
      - 默认展示：子分类标签/胶囊按钮
      - 行为约束：未选主分类时“添加子类”不可用或隐藏；已选主分类时可点击“添加子类”进入分类编辑页（新增子分类）
    - 快捷设置条（固定在输入区上方）：
      - 支付方式：显示当前选中（如“支付宝/微信/现金”等），点击可切换
      - 日期：显示“今天 + 时间”（或具体日期时间），点击可修改
      - 模版：打开模版选择弹窗，弹窗内提供“设置”入口进入模版账单页；点击模版可套用（预填分类/金额/备注/支付方式/标签）
      - 标签：进入标签选择弹窗（多选），弹窗内提供“设置”入口进入标签管理页
    - 输入区（固定）：
      - 备注输入框（左侧）
      - 金额输入框（右侧）：与数字键盘联动，支持“+/-”计算表达式输入，并在输入框内展示实时计算结果预览
    - 数字键盘（固定底部区域）：
      - 数字键：0-9、点号
      - 功能键：删除、加号、减号
      - 操作键：再记（保存并清空保持页面不关闭）、记录（保存并关闭返回）
    - 滚动策略：
      - 顶部栏/快捷设置条/输入区/数字键盘固定
      - 仅“分类区 + 子分类区 +（必要时）列表区域”允许在中间区域滚动
  - 分类管理与编辑（新增，来自流程图）：
    - 入口一：记账页分类区标题行的“分类管理”按钮 → 打开“分类明细页”
    - 入口二：记账页子分类区的“添加子类”按钮（需已选主分类） → 打开“分类编辑页（新增子分类）”
    - 页面结构与跳转：
      - 记账页（二级页） → 分类明细页 → 分类详情页 → 分类编辑页
      - 分类明细页 → 添加分类（+）→ 分类编辑页
      - 分类详情页 → 添加子分类（+）→ 分类编辑页
      - 分类编辑页保存后返回上一层页面，并同步刷新列表与当前选中展示
    - 通用规则（适用于分类明细/详情/编辑）：
      - 页面承载：作为记账二级页内的全屏页面栈（从记账页打开；返回回到记账页，不影响底部tab）
      - 账本范围：所有操作仅作用于“当前账本”的分类数据
      - 类型范围：分类按支出/收入分组；分类明细页可切换支出/收入并刷新列表
      - 刷新策略：新增/编辑/删除成功后，返回上一页时列表需刷新；记账页分类网格也需同步刷新
      - 名称展示：分类名称最多1行省略；避免长文本撑破布局
    - 分类明细页（Category List）：
      - 顶部栏：返回、标题“分类明细”、更多菜单（可选）
      - 顶部筛选：支出/收入分段切换（默认继承记账页当前类型）
      - 列表内容：按当前类型展示“主分类”列表（图标+名称）
      - 列表排序（v1.2默认）：按创建时间倒序；内置默认分类固定在前（或以名称排序稳定展示）
      - 行为：
        - 点击某主分类行（或图标）→ 打开“分类详情页”
        - 点击“编辑”→ 打开“分类编辑页（编辑主分类）”
        - 点击“删除”→ 二次确认；删除成功后从列表移除（删除受规则约束）
      - 添加入口：右下角浮动“+”按钮 → 打开“分类编辑页（新增主分类）”
    - 分类详情页（Category Detail）：
      - 顶部栏：返回、标题“分类详情”、更多菜单（可选）
      - 内容区：
        - 顶部展示当前主分类信息（图标+名称）
        - 子分类列表：展示该主分类下的二级分类（图标+名称）
      - 行为：
        - 点击主分类“编辑”→ 打开“分类编辑页（编辑主分类）”
        - 点击子分类“编辑”→ 打开“分类编辑页（编辑子分类）”
        - 点击子分类“删除”→ 二次确认；删除成功后从列表移除（删除受规则约束）
      - 添加入口：右下角浮动“+”按钮 → 打开“分类编辑页（新增子分类，parentId=当前主分类）”
    - 分类编辑页（Category Edit，新增/编辑共用）：
      - 顶部栏：返回、标题“分类编辑”、右侧“保存”
      - 基础字段：
        - 类型：支出/收入（新增时可选；编辑时默认锁定为原类型）
        - 名称：分类名称输入（必填，长度限制；建议≤10字/≤20字符）
        - 图标：支持从内置图标库选择（与当前分类网格保持一致；保存后在分类网格/列表统一展示）
        - 颜色：支持从预置色盘选择
        - 父级：仅新增/编辑子分类时展示（显示所属主分类且不可为空）
      - 校验规则（v1.2默认）：
        - 名称不可为空、不可全空格
        - 同一账本、同一类型、同一父级下名称不可重复
        - 子分类必须归属一个主分类（parentId 必填）
      - 保存规则：
        - 保存成功后返回上一页，并刷新显示
        - 从记账页“添加子类”进入新增子分类：保存后返回记账页并自动选中新建子分类
    - 删除规则（v1.2默认）：
      - 若分类/子分类已被任何记录引用：不允许删除，提示“该分类已有记录使用，无法删除”
      - 若主分类存在子分类：不允许删除，提示需先处理子分类
- 记账页标签管理（新增，来自流程图）：
  - 入口：记账页快捷设置条点击“标签”打开标签选择弹窗（多选）
  - 弹窗能力：
    - 展示标签列表（胶囊按钮），支持多选/取消多选
    - 顶部右侧提供“设置”按钮，进入“标签管理页”
  - 标签管理页（Tag Manage）：
    - 顶部栏：返回、标题“标签管理”
    - 列表内容：展示当前账本标签列表（名称，1行省略）
    - 行为：
      - 编辑：点击“编辑”打开“修改标签”对话框，支持修改标签名称（校验重名/空值）
      - 删除：点击“删除”二次确认；删除成功后从列表移除，并从所有记录的 tags 中移除该标签
    - 添加入口：右下角浮动“+”按钮 → 打开“添加标签”对话框，输入名称后新增
  - 弹窗与管理页联动：
    - 从标签管理页返回标签选择弹窗后，标签列表需刷新
    - 若当前已选标签在管理页被删除：返回后应自动从已选列表中移除
- 记账页模版设置（新增，来自截图）：
  - 入口：记账页快捷设置条点击“模版”打开模版选择弹窗
  - 弹窗能力：
    - 展示模版列表项（图标/名称/金额等信息），点击可套用到当前记账表单
    - 顶部右侧提供“设置”按钮，进入“模版管理页”
  - 模版管理页（Template Manage）：
    - 页面承载：作为记账二级页内的全屏页面栈（从模版弹窗进入；返回回到模版弹窗）
    - 顶部栏：返回、标题“模板管理”
    - 空状态（首次进入/无模版时）：
      - 展示空提示（如“暂无模板”）与“创建模板”按钮
      - 点击“创建模板”进入“模版账单页（创建）”
    - 列表状态（有模版时）：
      - 列表展示：模版名称（1行省略）
      - 每行操作：编辑、删除
      - 新建入口：右下角浮动“+”按钮，进入“模版账单页（创建）”
    - 行为约束：
      - 删除：二次确认；删除后仅移除该模版，不影响已有记录数据
  - 模版账单页（Template Bill，创建/编辑复用）：
    - 入口：
      - 从模版管理页点击“创建模板”或“+” → 创建
      - 从模版管理页点击某条模版“编辑” → 编辑
    - 顶部栏：返回、标题“模板账单”
    - 字段与交互（参考截图）：
      - 账单名称：页面顶部输入框（必填）
      - 金额：点击进入金额输入（可复用记账页金额键盘能力或输入框）；默认 0
      - 备注：点击进入备注输入
      - 分类：点击打开分类选择弹层/面板（同记账页分类选择，支持子分类）
      - 标签：点击打开标签选择弹窗（同记账页标签选择）；弹窗内仍可进入标签管理
      - 账户：点击选择支付方式（现金/微信/支付宝等）
      - 账本：展示当前账本（v1.2默认锁定为当前账本，不在该页切换）
    - 底部操作：
      - 保存：保存成功后返回模版管理页并刷新列表
      - 删除：仅编辑态显示；删除成功后返回模版管理页并刷新列表
  - 联动规则：
    - 从模版管理页返回模版弹窗后，模版列表需刷新
    - 模版保存后可立即在模版弹窗中被套用到当前记账表单
- `[ ]` 统计页面：展示图表统计、周期切换（周/月/年）
- `[ ]` 报表页面：报表概览、筛选与明细查看
  - 入口形态（v1.2）：底部导航“报表”tab进入
  - 核心查看流程：选择周期（周/月/年）与类型（支出/收入）→ 查看分类排行 → 点击分类查看明细列表
- `[ ]` 账户页面：展示净资产、总资产、总负债、账户列表
- `[ ]` 记录详情页：查看单条记录的详细信息
  - 记录详情页提供“编辑/删除”等操作入口（从首页列表移除直接编辑/删除按钮）

### 2.2 用户交互
- 底部导航栏切换页面
- 记账入口为二级页面：从任意页签点击底部中间按钮打开；关闭后回到打开前页签
- 记账时点击分类图标快速选择
- 记账页分类管理：从分类区进入分类明细/详情/编辑；可新增分类与新增子分类
- 记账页标签管理：标签弹窗内提供设置入口，进入标签管理页（新增/修改/删除）
- 记账页模版设置：模版弹窗内提供设置入口，进入模版管理页；创建/编辑进入模版账单页
- 支持快捷选项（今天、昨天、模版等）
- 记账页滚动：仅中间的分类/子分类区域滚动，键盘与输入区保持固定
- 首页记录列表：点击单条记录进入“记录详情页”
- 记录详情页：顶部更多菜单（或操作按钮）唤起操作面板，包含“编辑/删除”等
- 报表页入口与筛选交互：从底部导航进入报表页，支持周/月/年与支出/收入切换，并可下钻到分类明细
- 现代化的卡片式布局
- 流畅的页面切换动画

## 3. 验收标准
- `1.` 底部导航栏正常工作，可切换四个页面（首页、报表、统计、账户），并提供“记账”二级入口
- `1.1.` 点击底部“记账”按钮打开记账二级页；关闭后返回打开前的页签且保持该页原状态
- `1.2.` 从记录详情页点击“编辑”进入记账二级页；保存或关闭后返回打开前的页签且保持该页原状态
- `2.` 首页展示本月统计概览（收入、支出、结余）
- `2.1.` 首页顶部栏固定在顶部，展示当前账本与页面标题
- `2.2.` 首页记录列表按日期分组展示，并展示当日收入/支出/结余小计
- `2.3.` 首页列表点击单条记录进入详情页；编辑/删除操作在详情页完成
- `2.4.` 首页搜索状态下，分组与小计按“过滤后结果”计算
- `2.5.` 首页“加载更多”后，分组与小计随当前已展示数据更新
- `2.6.` 首页空记录/空搜索结果有明确提示与引导
- `2.7.` 从记录详情返回首页列表时，保持原滚动位置
- `3.` 记账页面支持支出/收入切换，分类以网格形式展示
- `4.` 记账页面支持快捷选项（今天、昨天、模版、必要消费等）
- `4.1.` 记账页顶部栏固定，包含账本入口、支出/收入切换与关闭按钮
- `4.2.` 记账页仅分类/子分类区域可滚动，输入区与数字键盘固定不跟随滚动
- `4.3.` 记账页“再记”保存后不关闭页面并保留最近一次选择（类型/分类/支付方式/标签等可复用）
- `4.4.` 记账页“添加记录”保存后关闭并返回打开前页签
- `4.5.` 记账页“分类管理”可进入分类明细页，并可进入分类详情与分类编辑页
- `4.6.` 分类明细页支持支出/收入切换，并可新增/编辑/删除主分类（删除受规则约束）
- `4.7.` 分类详情页支持新增子分类、编辑子分类、删除子分类（删除受规则约束）
- `4.8.` 分类编辑页支持新增/编辑分类与子分类；保存后返回并刷新；从记账页新增子分类后自动选中
- `4.9.` 记账页金额输入支持“+/-”计算表达式输入，并可实时预览计算结果；保存时按计算结果入账
- `4.10.` 记账页标签弹窗提供“设置”入口进入标签管理页；支持新增/修改/删除标签并与弹窗联动刷新
- `4.11.` 记账页模版弹窗提供“设置”入口进入模版管理页；支持创建/编辑/删除模版并与弹窗联动刷新
- `5.` 统计页面展示图表统计（如饼图），支持周/月/年周期切换
- `6.` 账户页面展示资产信息（净资产、总资产、总负债）和账户列表
- `7.` 可以查看记录详情
- `8.` 界面美观，操作流畅，页面切换动画流畅
- `9.` 移动端适配良好，所有按钮可点击区域≥44px
- `10.` 报表页可进入并完成核心查看流程
- `10.1.` 报表页入口在底部导航“报表”tab
- `10.2.` 报表页支持周/月/年与支出/收入切换，切换后数据同步更新
- `10.3.` 报表页支持分类排行展示，点击分类可查看该分类明细列表

## 4. 范围边界
### In（包含）
- 底部导航栏实现（四tab：首页/报表/统计/账户 + 中间主按钮“记账”二级入口）
- 四个主Tab页面的UI重构（首页/报表/统计/账户）与记账二级页面UI重构
- 首页/记账/统计/报表页面的UI布局与交互调整（补充需求）
- 报表页（v1.2）：作为底部导航Tab，承担分类占比环形图与明细列表
- 分类网格布局
- 分类管理相关页面（分类明细/分类详情/分类编辑）与从记账页的跳转
- 标签管理页面（标签管理页）与从记账页标签弹窗的跳转
